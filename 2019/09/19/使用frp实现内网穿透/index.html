使用frp实现内网穿透

发表于 2019-09-18 更新于 2020-06-19 分类于 Linux 阅读次数： 阅读次数：
本文字数： 1.8k 阅读时长 ≈ 2 分钟
准备工作

  1. 一台公网服务器(腾讯云, 阿里云 … 只要有外网ip的都可以)

  2. 一台内网机器(没有外网ip, 只有内网ip, 但是可以访问外部网络)

  1. 下载frp软件

    • GitHub地址(可以获取最新版): https://github.com/fatedier/frp/releases

    • 网盘下载(0.29.0版本): 链接:https://pan.baidu.com/s/1muHqT-g-Ocn6aTKg5l_cHQ 密码:gefm

公网服务器配置
  • 将frp软件传到服务器上(这里使用腾讯云)
  • 腾讯云安全组配置, 开放端口:
    • 7000 (frp 服务端绑定端口)
    • 9000 (frp http 访问端口)
    • 6000 (frp ssh 连接端口)
  • 编辑frps.ini文件
  • 启动frp 服务端
内网服务器配置
  • 将frp软件传到内网机器上
  • 编辑frpc.ini文件
  • 启动frp 客户端
使用第三台电脑进行访问
  • ssh 连接
  • nginx web 访问
    http://nginx.dazhi.info:9000/
  • nexus web 访问
    http://nexus.dazhi.info:9000/
结尾

  1. 至此就完成了内网穿透

  2. 更多配置访问frp官方文档.

    https://github.com/fatedier/frp/blob/master/README_zh.md
